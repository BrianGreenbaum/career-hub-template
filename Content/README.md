# Content

Writing and publishing workflow — ideas, drafts, and published posts across whatever platforms you care about (LinkedIn, Substack, Twitter/X, personal blog, etc.).

## How it works

```
Content/
├── CLAUDE.md          # Workflow instructions
├── ideas.md           # Running list of ideas, tagged by platform
└── {platform}/
    ├── drafts/        # Drafts in progress
    └── published.md   # Log of published pieces
```

Platform subfolders are created as you go — no need to create one for a platform you don't use.

## Workflow

1. **Capture ideas** in `ideas.md` — tag them with the intended platform.
2. **Draft** in the platform's `drafts/` folder.
3. **Publish**, then log the piece in `published.md` with date, link, and a quick note on how it landed.

`/update` will surface content seeds from your weekly journaling and add them here automatically (if you've opted into journaling).
